#### To start the server:
- `cd api-server`
- `npm install`
- `npm start`

### To start the frontend:
- `cd frontend`
- `npm install`
- `npm start`

You can also specify an api domain as an enviroment variable (`REACT_APP_API_DOMAIN`)
**Example:** `env REACT_APP_API_DOMAIN='http://localhost:3001' npm run start`
